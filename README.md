#Oque é este projeto?
   Este projeto é um SaaS simples de relógio atual em 3 cidades de SP com integração a uma api gratuita de clima

#O porque deste projeto?
   Ele foi feito com o foco de adquirir mais conhecimento com códigos 

#Oquem tem nesse projeto?
   Este projeto contém três tecnologias principais para o mercado de trabalho:
      FrontEnd (Html + Css + JavaScript)
      BackEnd (JavaScript puro)
      Mobile (React Native + Expo)
   
#Como você pode utilizar o projeto?
   Você pode utlizar esse projeto de três formas:
      FronEnd, BackEnd > Será necessário você pegar uma api_key no website: "https://home.openweathermap.org/"
      Mobile > Basta instalar o aplicativo em forma de apk em seu android! Não será necessário api_key neste app

#O usuário deve pagar algo?
   Não necessariamente, pois caso ele usar muito a sua api_key, será necessário pagar pelo rate limit da api, mas este projeto e a api e totalmente gratuita, só deve se atentar ao uso da api
